----------

Gammal info. 


1. Se till att neo4j informationen stämmer med informationen i (api/.env). Se till att URI är
inställd på bolt://localhost istället för bolt://neo4j.
2. Installera Neo4j från internet.
3. Ladda ner plugin till neo4j som kallas APOC (tryck på din Graph -> Manage -> Plugins)
4. Installera Node.js

För att starta backend:
	gå till /api
	skriv "npm start" och enter.
	
Om "npm" inte funkar så se till att node är installerat och att din PATH variabel är korrekt.

Exempel på hur man kan skicka queries och mutations finns i exempel.py!
